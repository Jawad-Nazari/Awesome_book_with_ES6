# 📖 [Awsome_Book_with_ES6] <a name="about-project"></a>

**Awesome Books With ES6!** is my third project at Microverse, in this project I will create a personal book collection when the user can add or delete books and this information will be preserved in the local storage system.This project is refactored using ES6 and moduled to the classes and also luxon librarary is used to display the current time.
